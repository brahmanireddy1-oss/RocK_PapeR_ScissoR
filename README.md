
# Rock Paper Scissors Game

A simple interactive **Rock Paper Scissors game** built using **Python and Flask** with an HTML interface. The player competes against the computer, which randomly selects its move.

## Features

* Interactive web interface
* Computer generates random choices
* Displays player and computer moves
* Shows game result (Win / Lose / Draw)
* Simple and user-friendly design

## Technologies Used

* Python
* Flask
* HTML
* CSS

## Project Structure

```
RockPaperScissor
│
├── app.py
└── templates
     └── index.html
```

## How to Run

1. Clone the repository

```
git clone https://github.com/brahmanireddy1-oss/RockPaperScissor.git
```

2. Install Flask

```
pip install flask
```

3. Run the application

```
python app.py
```

4. Open the browser and go to

```
http://127.0.0.1:5000
```

## How the Game Works

* The player selects **Rock, Paper, or Scissor**.
* The computer randomly selects a move.
* The game compares both selections and determines the winner.
